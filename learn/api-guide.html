Code Examples

Copy-paste ready examples in your favorite language.

# Upload and sign a document
curl -X POST https://api.brillsign.com/v1/sign \
  -H "Authorization: Bearer $BRILLSIGN_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "document_id": "doc_abc123",
    "signers": [
      { "email": "alice@example.com", "role": "signer" },
      { "email": "bob@example.com", "role": "approver" }
    ],
    "encryption": "aes-256-gcm"
  }'
# pip install brillsign
from brillsign import Client

client = Client(api_key="sk_live_...")

# Upload and sign
result = client.documents.sign(
    document_id="doc_abc123",
    signers=[
        {"email": "alice@example.com", "role": "signer"},
        {"email": "bob@example.com", "role": "approver"}
    ],
    encryption="aes-256-gcm"
)

print(result.signature_id)  # sig_7f3k9...
// npm install @brillsign/sdk
const { BrillSign } = require('@brillsign/sdk');

const client = new BrillSign('sk_live_...');

// Upload and sign
const result = await client.documents.sign({
  documentId: 'doc_abc123',
  signers: [
    { email: 'alice@example.com', role: 'signer' },
    { email: 'bob@example.com', role: 'approver' }
  ],
  encryption: 'aes-256-gcm'
});

console.log(result.signatureId); // sig_7f3k9...
// Maven: com.brillsign:sdk:1.0.0
import com.brillsign.BrillSignClient;
import com.brillsign.models.*;

BrillSignClient client = new BrillSignClient("sk_live_...");

// Upload and sign
SignResult result = client.documents().sign(
    SignRequest.builder()
        .documentId("doc_abc123")
        .addSigner("alice@example.com", "signer")
        .addSigner("bob@example.com", "approver")
        .encryption("aes-256-gcm")
        .build()
);

System.out.println(result.getSignatureId()); // sig_7f3k9...

Core Endpoints

The essential API endpoints you'll use most frequently.

POST /v1/sign

Initiate a signing request for one or more signers on an uploaded document.

GET /v1/verify/{id}

Cryptographically verify a signature using its unique ID and audit proof.

POST /v1/documents/upload

Upload an encrypted document for signing. Supports PDF, DOCX, and PNG formats.

GET /v1/documents/{id}

Retrieve document metadata, signing status, and download links.

POST /v1/webhooks

Register webhook URLs to receive real-time notifications on signing events.

DELETE /v1/documents/{id}

Permanently delete a document and all associated signatures from the system.
