Search "BrillSign" in the HubSpot App Marketplace. Click Install — no code, no technical setup required. Works with HubSpot free through Enterprise.

Connect Your BrillSign Account

Authorize BrillSign with your HubSpot portal via OAuth. Your deals, contacts, and document templates sync automatically.

3

Map Deal Stages to Signing Events

Configure which HubSpot deal stage maps to "Contract Sent" and "Contract Signed." BrillSign will advance the stage on each signing event automatically.

4

Create or Import Contract Templates

Build signing templates in BrillSign using HubSpot merge variables — {{company}}, {{amount}}, {{contact.email}} — for automatic data fill on each send.
