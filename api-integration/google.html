How It Works

From Drive File to Signed Doc in 3 Clicks.

1

Right-click in Drive

Select any file in Google Drive and click "Send for Signing" from the context menu.

2

Signers Sign On Any Device

Signers receive a branded email. They sign on desktop or mobile with no BrillSign account needed.

3

Signed Doc Saved to Drive

The completed document auto-saves back to Drive. You get a notification in Gmail. No action needed.

Quick Setup

Install from Google Workspace Marketplace.

1

Install from Marketplace

Search "BrillSign" in the Google Workspace Marketplace. Install for yourself or deploy org-wide as an admin.

2

Authorize with Google

Grant BrillSign access to Drive and Gmail via Google's standard OAuth flow. BrillSign only requests the scopes it needs.

3

Connect Your BrillSign Account

Sign in or create your BrillSign account. Your Drive is now connected — right-click any file to start signing immediately.

4

Configure Default Folder

Set a default Drive folder where signed documents get saved. Supports shared drives and personal My Drive.
